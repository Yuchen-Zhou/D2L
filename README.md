## 简介
这个仓库用于记录[动手学深度学习](https://zh-v2.d2l.ai/)这本书的笔记。这本书主要偏向于编程实践，会用到TensorFlow、PyTorch两个框架。一般来说我们使用conda环境

同时你还需要自己安装d2l
